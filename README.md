# 📖 Sobre o Projeto

O **EcoFila** é um sistema web desenvolvido com o propósito de otimizar o gerenciamento de filas de atendimento por meio da tecnologia. A proposta do projeto consiste em oferecer uma plataforma intuitiva, organizada e eficiente, capaz de reduzir o tempo de espera e melhorar a experiência tanto dos usuários quanto dos responsáveis pelo atendimento.

Em diversos estabelecimentos, como clínicas, instituições públicas, universidades e empresas, a organização das filas ainda representa um desafio. Longos períodos de espera, falta de informações e dificuldades no gerenciamento impactam diretamente a qualidade do atendimento.

Pensando nesse cenário, o EcoFila foi desenvolvido como uma solução tecnológica que busca automatizar esse processo, promovendo maior organização, transparência e praticidade.

Além do seu propósito funcional, o projeto também representa a aplicação prática dos conhecimentos adquiridos durante a formação acadêmica, envolvendo conceitos de desenvolvimento web, design de interfaces e organização de software.

---

# 🎯 Objetivos

O projeto possui como principais objetivos:

* Desenvolver uma plataforma moderna para gerenciamento de filas;
* Organizar o fluxo de atendimento de maneira eficiente;
* Reduzir o tempo de espera dos usuários;
* Melhorar a experiência durante o atendimento;
* Aplicar boas práticas de desenvolvimento Front-end;
* Desenvolver uma interface intuitiva e acessível;
* Promover uma navegação simples e agradável.

# ▶️ Como Executar

1. Clone este repositório.

```bash
git clone https://github.com/marareiaa/EcoFila.git
```

2. Acesse a pasta do projeto.

```bash
cd EcoFila
```

3. Abra o arquivo **index.html** em seu navegador.

Ou utilize a extensão **Live Server** no Visual Studio Code para uma experiência de desenvolvimento mais prática.
